# wfauto2

世界弹射物语/弹射世界 的手机端护肝宝，基于 Auto.js Pro 9 开发。

## 项目状态

持续开发中，目前已实现：

- 服务器支持：
  - [x] 国服
- 功能支持：
  - [x] 自动挂铃铛
    - [x] 白嫖模式（不续战铃铛）
    - [x] 自动使用体力药水
    - [x] *过滤铃铛（测试中）*
  - [x] ~~自动单人~~（目前国服已经实装单人副本自动续战）
    - [ ] 自动地狱本
    - [ ] 自动活动单人本
